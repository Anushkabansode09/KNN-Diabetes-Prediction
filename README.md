# KNN Diabetes Prediction

## Overview
This project predicts whether a person has diabetes using the K-Nearest Neighbors (KNN) algorithm.

## Steps Performed
- Data Exploration
- Data Preprocessing
- Model Building (KNN)
- Model Evaluation

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Result
Achieved good accuracy using KNN after normalization.
